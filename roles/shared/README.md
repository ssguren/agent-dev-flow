# Shared Commands — 共享命令

这里存放多个角色共用的命令。各角色的 `profile.md` 中会引用这些命令。

| 命令 | 主要使用角色 | 说明 |
|------|------------|------|
| `commands/review-pr.md` | Tech Lead / Backend / Frontend | PR 规范检查 + 上下文摘要，Reviewer 在人工 Review 前运行 |
| `commands/draft-adr.md` | Tech Lead / Backend | 起草架构决策记录，留白 Decision 节给人填 |
| `commands/write-pr-desc.md` | Backend / Frontend | 基于 git diff 生成标准 PR 描述 |
| `commands/confirm-gap.md` | PM / Tech Lead | 人工确认 Gap 决策，写入 gap-registry.md |
